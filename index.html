Harry Potter Universe - Detalles Técnicos

Objetivo: Crear una interfaz dinámica que permita explorar la base de datos de personajes de Harry Potter.

  • Tecnologías: HTML5, CSS3, JavaScript (ES6+), Fetch API.
  • Desafíos: Implementar un buscador en tiempo real y filtros por "Casa" (Gryffindor, Slytherin, etc.) sin recargar la página.
  • Solución: Uso de Array.filter() y manipulación directa del DOM para renderizar tarjetas optimizadas.

Rick & Morty Explorer - Detalles Técnicos

Objetivo: Consumir una API pública masiva y presentar la información de manera paginada y eficiente.

  • Tecnologías: JavaScript Asíncrono (Async/Await), JSON, CSS Grid.
  • Arquitectura: Estructura modular separando la lógica de llamada a la API de la lógica de renderizado.
  • Funcionalidad Clave: Paginación dinámica que carga nuevos personajes bajo demanda.
